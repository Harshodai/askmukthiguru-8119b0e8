## Backend (Railway FastAPI — separate repo)

### CORS — required for prod chat to work
Add to your FastAPI `main.py`:

```python
from fastapi.middleware.cors import CORSMiddleware

app.add_middleware(
    CORSMiddleware,
    allow_origins=[
        "https://askmukthiguru.lovable.app",
    ],
    allow_origin_regex=r"^https://([a-z0-9-]+\.)?lovable\.app$",
    allow_credentials=True,
    allow_methods=["*"],
    allow_headers=["*"],
    expose_headers=["*"],
)
```

`allow_origin_regex` covers `id-preview--*.lovable.app` and any other subdomains.
