# Cheatsheet: Decision Tables, Comparison Matrices, and Quick Reference Rules

## **Decision Tables**
A decision table is a structured way to represent decisions based on multiple conditions. It helps in evaluating different scenarios systematically.

### Components of a Decision Table:
1. **Conditions**: The factors that influence the decision.
2. **Actions**: The possible outcomes or decisions.
3. **Rules**: The logic connecting conditions to actions.

### Example Decision Table:

| Conditions \ Actions | Condition 1 | Condition 2 | Action |
|-----------------------|-------------|-------------|--------|
| Condition 1 Value 1   | Yes         | No          | A      |
| Condition 1 Value 2   | No          | Yes         | B      |

---

## **Comparison Matrix**
A comparison matrix is a tool used to compare two or more items based on predefined criteria. It helps in evaluating the strengths and weaknesses of each item.

### Components of a Comparison Matrix:
1. **Items**: The objects being compared.
2. **Criteria**: The aspects used for comparison.
3. **Scoring**: A numerical value assigned to each criterion.

### Example Comparison Matrix:

| Criteria \ Items | Item 1 | Item 2 |
|-------------------|--------|--------|
| Criterion 1       | 8      | 7      |
| Criterion 2       | 6      | 9      |

---

## **Quick Reference Rules**
Quick reference rules are concise guidelines that provide immediate solutions or best practices for common situations.

### Example Quick Reference Rules:
1. **Rule 1**: Always start the project documentation phase early to avoid delays.
2. **Rule 2**: Use a Gantt chart for visualizing project timelines and dependencies.
3. **Rule 3**: Conduct a risk assessment before starting any major task.

---

## **Summary**
- **Decision Tables** help structure decisions based on multiple conditions.
- **Comparison Matrices** allow for systematic comparison of items based on criteria.
- **Quick Reference Rules** provide quick solutions or best practices for common scenarios.

This cheatsheet serves as a handy reference guide for practitioners to apply these tools effectively in their work.
