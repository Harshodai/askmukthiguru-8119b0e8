# Glossary

Here is an organized glossary based on the provided chapters:

- **AI** — Artificial Intelligence, a branch of computer science focused on creating systems that can perform tasks requiring human intelligence. (Ch12)
  
- **LLM** — Large Language Model, a type of AI system that understands and generates human language. (Ch1)

- **Tokens** — The basic units in Natural Language Processing (NLP) used to represent words or subwords in models like BERT or GPT. (Ch2)

- **Context Window Size** — The amount of context a model can consider when processing text, crucial for tasks requiring understanding of surrounding information. (Ch2)

- **Pre-trained Models** — Models like BERT and GPT that have been trained on large datasets to perform various NLP tasks before being fine-tuned for specific applications. (Ch2)

- **API** — Application Programming Interface, a set of protocols, tools, and definitions used to interact with a software application. (Ch3)
  
- **Rate Limits** — Restrictions in place to prevent abuse or overuse of an API by limiting the number of requests from a single source. (Ch3)

- **RESTful APIs** — A set of protocols that define how resources can be accessed, updated, and deleted using HTTP methods like GET, POST, PUT, etc. (Ch3)

- **Training Data** — The dataset used to train an LLM, where the model learns patterns from this data. (Ch5)

- **Annotation** — The process of labeling or marking up data to provide signals for training models, such as in sentiment analysis. (Ch4)

- **Data Pipelines** — Systems that collect, process, and prepare raw data into a format suitable for training machine learning models. (Ch4)

- **Fine-tuning** — The process of adjusting pre-trained models to improve their performance on specific tasks or datasets. (Ch5; Ch7)

- **Overfitting** — A situation where a model performs well on training data but poorly on new, unseen data due to excessive complexity. (Ch5)

- **Tokenization** — The division of text into tokens to facilitate processing by NLP models, allowing them to understand and manipulate language effectively. (Ch2; Ch5)

- **Generative AI** — A subset of AI focused on generating content such as text, images, or music using LLMs. (Ch6)

- **NLP** — Natural Language Processing, the field concerned with interactions between humans and computers via language, including tasks like translation, summarization, and question answering. (Ch2; Ch3; Ch4; Ch5; Ch7; Ch8; Ch10; Ch11; Ch12)

- **Model Serving** — The deployment of pre-trained models ready for use in applications, ensuring quick access and inference times. (Ch6; Ch8; Ch9)

- **Customization** — Adapting LLMs to meet specific project or application needs by modifying their parameters or configurations. (Ch8)

- **API Integration** — The process of embedding model outputs into APIs to enable seamless integration with external systems and services. (Ch8)

- **Embedding** — Representing words, phrases, or other text data in a high-dimensional numerical space for efficient processing by machine learning models. (Ch6; Ch8)

- **OpenAI** — A company known for developing advanced AI models like GPT, which has significantly impacted the LLM landscape. (Ch9)

- **Retraining** — The process of fine-tuning an existing model to adapt it to new tasks or datasets, often involving reusing pre-trained weights. (Ch9)

- **Code Generation Tools** — Software tools that automatically generate code for specific tasks, leveraging AI models to produce efficient and effective solutions. (Ch10)

- **Edge Computing** — A computing paradigm where processing occurs near the source of data generation, reducing latency and bandwidth requirements. (Ch11)

- **Raspberry Pi** — A low-cost, tiny computer with significant computing power used for various applications, including deploying machine learning models due to its versatility and affordability. (Ch11)

- **Model Optimization** — Techniques aimed at improving the efficiency of LLMs for deployment on specific hardware or devices, ensuring optimal performance while minimizing resource usage. (Ch11)

This glossary provides a comprehensive overview of key terms related to large language models, their development, operations, and applications across various chapters.
