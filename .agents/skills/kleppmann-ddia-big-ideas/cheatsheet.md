### Cheatsheet: Decision Tables, Comparison Matrices, and Quick Reference Rules  

#### **Decision Table**  
| **Criteria/Options** | **Outcome** |  
|-----------------------|-------------|  
| **Option 1**          | Result A   |  
| **Option 2**          | Result B   |  
| **...**               | ...         |  
| **Total Options (N)** |             |  

#### **Comparison Matrix**  
| **Criteria/Options** | **Option 1** | **Option 2** | **...** |  
|-----------------------|-------------|-------------|--------|  
| **Criteria 1**        | Value 1     | Value 2     | ...    |  
| **Criteria 2**        | Value A     | Value B     | ...    |  
| **...**               | ...         | ...         | ...    |  

#### **Quick Reference Rules**  
- Use a decision table when you have multiple options and criteria to evaluate.  
- Use a comparison matrix when you need to compare two or more items based on specific attributes.  
- Apply quick reference rules for simple decisions or repetitive tasks where speed is essential.
