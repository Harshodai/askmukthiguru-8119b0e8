## Core Frameworks & Mental Models

Here’s an alphabetical glossary of key terms from your provided chapters:

**Clustering** — A process of grouping data points into clusters based on similarity (Ch 1).  
**Embeddings** — A method to convert text into numerical representations for machine learning models (Ch 14).  
**Fine-tuning** — The process of adjusting model parameters to improve performance on a specific task (Ch 5, Ch 26, Ch 28).  
**Global gross** — A measure of global box office success, e.g., "The film had a worldwide gross" (Ch 20).  
**Multimodal large language model** — A model that processes and understands both text and other data types like images or audio (Ch 16).  
**Prompt engineering** — The practice of crafting effective prompts to guide AI outputs (Ch 15).  
**Relevance scoring** — A method to determine the relevance of retrieved information (Ch 10).  
**Text classification** — Categorizing text into predefined classes (Ch 11).  
**Text embedding** — Representing text as dense vectors for efficient processing (Ch 14).  
**Text generation** — Creating new text content using AI models (Ch 28).  
**Text searchability** — The ability to find relevant information within a dataset (Ch 10).  
**Text classification** — Categorizing text into predefined classes (Ch 11).  
**Text embedding** — Representing text as dense vectors for efficient processing (Ch 14).  
**Text generation** — Creating new text content using AI models (Ch 28).
