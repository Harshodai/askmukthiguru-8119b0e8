# Chapter 6: Word Embeddings

## Core Idea
Word embeddings are numerical representations of text that enable machines to understand semantic relationships and perform tasks like search, recommendation systems, and text summarization.

## Frameworks Introduced
- **Word2Vec**: A neural network-based algorithm trained on text data to produce word embeddings.  
  - When to use: For creating dense vector representations of words based on their co-occurrence in text.
  - How: Trains a neural network to predict neighboring words and learns their embeddings through contrastive loss.

## Key Concepts
- **Text Embeddings**: Numerical vectors representing entire pieces of text, generated by averaging or pooling word embeddings.  
- **Word2Vec**: Generates word embeddings using skip-gram architecture with negative sampling.  
- **Skip-Gram**: A method for learning word embeddings by predicting context words given a target word.

## Mental Models
- Use word embeddings when you need to represent text numerically for machine learning tasks like classification, clustering, or recommendation systems.

## Anti-patterns
- **Capitalization Issues**: Over-reliance on capitalization can lead to poor model performance in language modeling tasks.  
  - Why it Fails: Capitalization introduces unnecessary variability and makes models less effective at understanding linguistic patterns.

## Code Examples
```
# Example of Word2Vec using Gensim
from gensim.downloader import api
model = api.load("glove-wiki-gigaword-50")

# Get the embedding for a word
print(model['king'])  # Outputs: array([-3.169e-04,  8.279e-05, ...,  1.000e+00])
```

This code demonstrates loading pre-trained word embeddings and retrieving the vector for a specific word.

## Key Takeaways
1. Word embeddings are essential for converting text into numerical representations that capture semantic meaning.
2. The choice of embedding model depends on the specific application (e.g., BPE for NLP tasks vs. CLIP for vision-language models).
3. Proper tokenization strategies, including handling capitalization and punctuation, are critical for effective modeling.
